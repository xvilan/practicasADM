# Prácticas de Análsis de datos multivariantes  

Este repositorio es una web para la entrega de prácticas de la asignatura de Análisis de datos multivariantes, del Máster de Ciencias Actuariales y Financieras (UC3M). 

Está creada combinando una plantilla de HTML5 UP con los ejercicios resultos en Rmarkdown.
