Bienvenido!

Esta es una web para la entrega de prácticas de la asignatura de Análisis de datos multivariantes, del Máster de Ciencias Actuariales y Financieras (UC3M). Es una web responsitiva creada con una plantilla HTML5 diseñada por AJ y distribuida libremente por HTML5 UP bajo licencia CCA 3.0. Sus características son simpleza, diseño ligero, codigo HTML5 y CSS sólido y totalmente responsitivo para escritorio, tablets y pantallas de móviles.

Resumen de los elementos utilizados en la web

El objetivo de esta web es mostrar los pasos realizados en cada práctica de una manera sencilla, limpia y didáctica. En esta sección se resumen los elementos que se utilizan.

  • Gifs demostrativos

    Los gifs están hechos mediante capturas de pantalla usando el progrma GifCam. Se usa para los procedimientos realizados en Microsot Excel y Real-Statisctics. Muestra algunos pasos para la configurar de Excel y el uso de las funciones de cálculo matricial.

  • Notebooks

    Los notebook están creados con R-Studio, que utiliza knitr y pandoc para exportar el código a multiples formatos, entre ellos .md y .html. En la parte superior derecha del Notebook (botón Code) se pued descargar el archivo .Rmd para ejecutar los scripts en R-Studio.

  • Código

    Tanto el código de R Markdown, como los comandos de Excel y Real-Statistics están disponibles en las prácticas. Desde esta página de inicio también se puede acceder a ellos. Se puede hacer un fork o descargar esta web dende github.

Programa de la Asignatura

El objetivo último de esta asignatura es proporcionar al estudiante conocimiento y comprensión de algunas técnicas de Análisis de Datos Multivariantes que le permitan medir relaciones entre variables con datos reales mediante el modelo más adecuado, dependiendo de la naturaleza de los datos y de la información que se quiera obtener. Concretamente, se aborda el aprendizaje de Análisis de Conglomerados y del Análisis Factorial, del Modelo de Regresión (considerando la presencia de variables explicativas cualitativas), así como, de los Modelos Lineales Generalizados.
