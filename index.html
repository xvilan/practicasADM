Elementos utilizados en esta web

El objetivo de esta web es mostrar los pasos realizados en cada práctica de una manera sencilla, limpia y didáctica. En esta sección se resumen los elementos que se utilizan.

  • Notebooks

    Los notebook están creados con R-Studio, que utiliza knitr y pandoc para exportar el código a múltiples formatos. Estas prácticas pueden descargarse en formato.Rmd y ejecutar los script en R-Studio.

  • Código

    Tanto el código de R Markdown, como los comandos de Excel y Real-Statistics están disponibles en las prácticas. Desde esta página de inicio también se puede acceder a ellos. Se puede hacer un fork o descargar esta web completa desde github.

Programa de la Asignatura

El objetivo último de esta asignatura es proporcionar al estudiante conocimiento y comprensión de algunas técnicas de Análisis de Datos Multivariantes que le permitan medir relaciones entre variables con datos reales mediante el modelo más adecuado, dependiendo de la naturaleza de los datos y de la información que se quiera obtener. Concretamente, se aborda el aprendizaje de Análisis de Conglomerados y del Análisis Factorial, del Modelo de Regresión (considerando la presencia de variables explicativas cualitativas), así como, de los Modelos Lineales Generalizados.
